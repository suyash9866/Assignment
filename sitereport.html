Report :

When I originally started at NAMI College for my bachelor's degree, I had no knowledge of programming languages or codes. Nonetheless, I chose this course because I am fascinated by coding and programming languages. When I first started taking Web classes, I had a hard time adjusting to the module (CSY1018) and programming languages. But, with the support of my friends and the module tutor, I gradually became more comfortable with the coding. I also viewed a lot of HTML and CSS videos, which were quite helpful. In this module, I used all of my time to its full extent. I would talk to my buddies or module tutor if I had any questions about the module.

Now it was time for my project, and we were all running out of time. I had a lot on my plate, and I still had a lot of work to do on my module's skill work. My mind was so frustrated that I was fatigued from all of these chores that I began to tackle them one by one. By the time I started to like my assignment work, I was having a lot of fun writing CSS and HTML code, and our respected sir had extended our time. Now our objective was to construct a total of 5 web pages and to visualize my work, I sketched my webpage designs and layout on plain paper. To begin, I design a homepage with a navigation bar that includes the following links: home, portfolio, blog, contact, and site report. For serially listing these components, I utilized the li> tag. All of them are linked in the navigation bar. For the link, I used the href tag. I used the nav> tag for navigation. I picked teal as the color combination since it looks good on any web page. I mentioned my name and welcome on the main page, therefore I used the span> tag to alter the color of specified text on my page. For the portfolio section, I've utilized the div> tag and [class=""] it with multiple names to identify it in CSS. This page will have placeholder photos that will eventually be filled with my work from term 2. An image of placeholder images has been added. I've also used gird to add columns to placeholders, and [border-radius] to add curves to the border's corners. The opacity is also lowered to make it more appealing to the eye. In order to match the color coordination, the backdrop color is also given according to the placeholder images. Padding is also done in accordance with the sizes required.

Now, for the blog section, I have three entries for the column to record my progress on the assignment, so I've done the same thing we do on the home page, with three columns written as follows: (phase 1, phase 2 & phase 3). Inside the box, I gave my contents correct padding and margin, and I utilized the same colors as the other pages (beige, light sea green, red), as well as a grid template column to divide my context regarding tasks.

Now for the contact section, I need to create a contact form for my webpage, therefore I've used form, label, input, and placeholder tags to create this contact form. They serve many purposes in various sectors. I also added my email to the submit button so they could contact me directly. I also used the "mailto" and "href" tags for the link.
Finally, for the report, I used three sectors of boxes on rows to present my images of HTML & CSS validations, image references, and some reports about my work, thus I almost utilized the same sorts of tags in this sector. I've also added my validation photographs to the site reports, and that concludes the project.

I was also required to create a hamburger icon as part of the work, but because our module instructor did not teach us how to do so, I was unable to do so.
